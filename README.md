## ⭐ 三大有趣发现

### 1. 频率统计特征（+6.4%）
- 计算站点/路线的会员使用比例
- 3个频率特征带来6.4%提升
- 所有特征中贡献最大

### 2. 最小特征集（8特征=97.8%）
- 仅8个特征达到完整模型97.8%性能
- 证明"less is more"
- 适合实时预测场景

### 3. 线性核最优
- LinearSVC > RBF核 > 多项式核
- 手工特征工程已提供足够非线性
- 特征工程比算法选择更重要
