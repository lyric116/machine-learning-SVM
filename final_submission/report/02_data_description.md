## 二、数据介绍

### 2.1 数据来源

本实验使用Capital Bikeshare公开发布的2025年1月至12月华盛顿都会区共享单车轨迹数据。原始数据共包含12个月，总计6,662,647条骑行记录。

### 2.2 原始数据字段

数据集包含以下13个原始字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| ride_id | 字符串 | 骑行唯一标识符 |
| rideable_type | 字符串 | 车辆类型（electric_bike/classic_bike/docked_bike） |
| started_at | 时间戳 | 骑行开始时间 |
| ended_at | 时间戳 | 骑行结束时间 |
| start_station_name | 字符串 | 起点站点名称 |
| start_station_id | 字符串 | 起点站点ID |
| end_station_name | 字符串 | 终点站点名称 |
| end_station_id | 字符串 | 终点站点ID |
| start_lat | 浮点数 | 起点纬度 |
| start_lng | 浮点数 | 起点经度 |
| end_lat | 浮点数 | 终点纬度 |
| end_lng | 浮点数 | 终点经度 |
| **member_casual** | **字符串** | **用户类型（member/casual）目标变量** |

### 2.3 数据抽样策略

考虑到原始数据规模较大（666万条）且存在类别不平衡问题，本实验采用**分层抽样**策略：

**抽样方案**：
- 按月份分层：保证12个月都有代表性样本
- 按类别分层：每月分别从member和casual中抽取相同数量样本
- 抽样规模：每月每类别抽取5,000条记录

**抽样流程**：
```
原始数据（6,662,647条）
    ↓ 按月份分组
月度数据（12个月）
    ↓ 按用户类型分组
每月每类别抽取5,000条
    ↓ 合并
抽样数据（120,000条）
    ↓ 数据清洗
最终样本（58,405条）
```

**最终样本分布**：
- 总样本量：58,405条
- 会员（member）：29,362条（50.3%）
- 散户（casual）：29,043条（49.7%）
- 时间跨度：2025年1-12月，每月均有样本
- 类别平衡：接近1:1，避免模型偏向
