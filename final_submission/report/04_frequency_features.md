#### 4.2.4 频率特征的创新价值

**频率特征定义**：
计算每个站点/路线被会员使用的比例

```python
# 起点站点的会员使用频率
member_count = df[df['member_casual']=='member']['start_station_id'].value_counts()
total_count = df['start_station_id'].value_counts()
start_station_freq = (member_count / total_count).fillna(0.5)
```

**为什么频率特征有效？**

1. **站点功能差异**：
   - 办公区站点：会员比例高（通勤）
   - 旅游景点站点：散户比例高（观光）
   - 居住区站点：会员比例较高（接驳）
   - 交通枢纽站点：混合使用

2. **路线模式**：
   - 固定通勤路线：会员占主导
   - 环线观光路线：散户占主导
   - 单程接驳路线：会员为主

3. **信息增益**：
   - 5个基础特征 → F1 = 0.5892
   - +3个频率特征 → F1 = 0.6271 (+6.4%)
   - **这是所有特征中贡献最大的一组**
