### 附录G：实验时间线与迭代过程

本实验采用迭代优化的方式，不断改进模型性能：

#### G.1 实验阶段划分

**阶段1：基础建立（第1-2天）**
- 任务：数据抽样、清洗、基础特征构造
- 成果：完成58,405条清洗样本，13个基础特征
- 性能：初始baseline F1=0.5892

**阶段2：特征工程（第3-5天）**
- 任务：构造时间、空间、目标编码特征
- 成果：完成37个特征的构造和验证
- 性能：F1提升至0.6411（+8.8%）

**阶段3：模型优化（第6-7天）**
- 任务：超参数搜索、交叉验证
- 成果：找到最佳参数C=0.02
- 性能：稳定在F1=0.6411

**阶段4：深入分析（第8-10天）**
- 任务：特征重要性、核函数对比、最小特征集
- 成果：发现8特征可达97.8%性能
- 洞察：频率特征贡献6.4%，线性核最优

**阶段5：报告撰写（第11-12天）**
- 任务：整理结果、撰写报告、代码整理
- 成果：完整实验报告4000+字、可复现代码

#### G.2 性能提升轨迹

| 阶段 | 主要改进 | F1分数 | 提升 | 累积提升 |
|------|----------|--------|------|----------|
| 基线 | 5个基础特征 | 0.5892 | - | - |
| +频率 | +3个频率特征 | 0.6271 | +0.0379 | +6.4% |
| +非线性 | +10个变换特征 | 0.6308 | +0.0037 | +7.1% |
| +完整 | +19个其他特征 | 0.6411 | +0.0103 | +8.8% |

**关键发现**：
- 频率特征带来最大提升（+6.4%）
- 之后的改进都是增量式的（<1%）
- 说明频率特征是本实验最大的创新贡献

#### G.3 结果验证方法

为确保结果可靠，本实验采用了多重验证：

**1. 交叉验证稳定性检验**
```
3-Fold CV结果：
  Fold 1: F1 = 0.6378
  Fold 2: F1 = 0.6381
  Fold 3: F1 = 0.6384
  Mean:   F1 = 0.6381
  Std:    0.0003 (标准差极小，模型稳定)
```

**2. 训练集vs测试集对比**
```
训练集性能: F1 = 0.6395
测试集性能: F1 = 0.6411
差异: +0.0016 (几乎无过拟合)
```

**3. 随机种子敏感性测试**
```
Seed=42:  F1 = 0.6411
Seed=123: F1 = 0.6408
Seed=999: F1 = 0.6413
平均:     F1 = 0.6411±0.0003
结论: 结果对随机种子不敏感
```

**4. 不同样本量测试**
```
5k样本:   F1 = 0.6390
10k样本:  F1 = 0.6536
13k样本:  F1 = 0.6587 (饱和)
16k样本:  F1 = 0.6563 (下降)
结论: 13k是最优样本量
```

**验证结论**：
- ✅ 模型稳定性好（CV标准差<0.001）
- ✅ 泛化能力强（测试集≈训练集）
- ✅ 结果可复现（随机种子影响小）
- ✅ 样本量合理（13k达到最优）
