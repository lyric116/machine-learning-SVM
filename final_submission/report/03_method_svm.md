## 三、实验方法

### 3.1 支持向量机（SVM）算法原理

#### 3.1.1 算法基本思想

支持向量机（Support Vector Machine, SVM）是一种基于统计学习理论的监督学习算法，核心思想是寻找一个最优超平面，使得两类样本之间的间隔最大化。

对于线性可分的二分类问题，SVM的目标是找到一个决策边界：

$$f(x) = w^T x + b$$

使得：
- 对于正类样本：$w^T x_i + b \geq +1$
- 对于负类样本：$w^T x_i + b \leq -1$

#### 3.1.2 软间隔SVM

现实数据往往不是完全线性可分的，因此引入软间隔SVM，允许部分样本违反间隔约束。优化目标为：

$$\min_{w,b,\xi} \frac{1}{2} ||w||^2 + C \sum_{i=1}^{n} \xi_i$$

约束条件：
$$y_i(w^T x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

其中：
- $w$：超平面法向量
- $b$：偏置项
- $\xi_i$：松弛变量，允许样本违反间隔
- $C$：正则化参数，控制间隔宽度与训练误差的权衡
  - $C$越大：对误分类惩罚越重，可能过拟合
  - $C$越小：允许更多误分类，可能欠拟合

#### 3.1.3 核函数

对于非线性问题，SVM通过核函数将样本映射到高维空间：

**1. 线性核（Linear Kernel）**：
$$K(x_i, x_j) = x_i^T x_j$$
- 适用于线性可分问题
- 计算速度快，不易过拟合

**2. RBF核（径向基函数）**：
$$K(x_i, x_j) = \exp(-\gamma ||x_i - x_j||^2)$$
- 可处理任意复杂的非线性边界
- 参数$\gamma$控制决策边界的复杂度

**3. 多项式核（Polynomial Kernel）**：
$$K(x_i, x_j) = (\gamma x_i^T x_j + r)^d$$
- degree $d$控制多项式阶数
- 适合捕捉特征间的高阶交互
