# Capital Bikeshare 用户类型分类实验报告

## 一、实验问题与目标

### 1.1 研究背景

共享单车作为城市公共交通的重要组成部分，在解决"最后一公里"出行问题上发挥着重要作用。Capital Bikeshare是美国华盛顿都会区的共享单车系统，拥有大量的用户和骑行记录。该系统主要服务两类用户：会员用户（member）和散户用户（casual）。准确识别用户类型对于运营优化、营销策略制定以及服务改进具有重要意义。

### 1.2 实验问题

本实验的核心问题是：**如何根据单次骑行的行为特征，使用支持向量机（SVM）算法准确预测用户类型？**

具体而言，给定一次骑行的时间、空间、车辆类型和站点信息，判断该用户是会员（member）还是散户（casual）。这是一个典型的二分类问题。

### 1.3 实验目标

本实验的主要目标包括：

1. **数据预处理**：完成变量筛选、缺失值处理、异常值过滤、数据标准化等预处理工作
2. **模型构建**：使用SVM算法构建用户分类模型，探索不同核函数和参数的效果
3. **性能评估**：按80%:20%划分训练集和测试集，使用准确率、F1分数等指标评估模型性能
4. **因素分析**：深入分析时间、空间、行为等不同因素对用户类型的影响强度
5. **创新发现**：探索特征工程的价值，寻找有趣的实验发现

### 1.4 实验创新点

1. **系统化特征工程**：从原始13个字段构造出37个有效特征
2. **频率统计特征**：创新性地引入站点和路线的会员使用频率特征
3. **最小特征集探索**：发现仅用8个特征即可达到完整模型95%的性能
4. **多核函数对比**：对比LinearSVC、RBF核和多项式核的实际效果
