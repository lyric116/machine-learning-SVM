## ⭐ 三大有趣发现

### 1. 频率统计特征的突破（+6.4%）

**创新**：计算每个站点/路线被会员使用的比例

```
基础5特征:  F1 = 0.5892
+频率3特征: F1 = 0.6271 (+6.4%) ⬆️ 最大贡献
+其他29个:  F1 = 0.6587 (+4.8%)
```

**原理**：不同功能区的用户构成差异
- 办公区站点：会员78%（通勤）
- 旅游景点：散户68%（观光）
- 地铁站：混合65%

### 2. 最小特征集（8特征=97.8%性能）

**发现**：仅8个特征即可达到完整模型97.8%的性能

```
1个特征(speed):        F1 = 0.5672 (86.1%)
8个特征(基础+频率):    F1 = 0.6271 (97.8%)
37个特征(完整):        F1 = 0.6587 (100%)
```

**意义**：
- 适合实时预测（快速）
- 适合移动端部署（轻量）
- 证明"less is more"

### 3. 线性核优于非线性核

**对比**：
- LinearSVC:  F1=0.6411 ✅ 最优（8秒）
- RBF核:      F1=0.6354（45分钟）
- Poly核:     F1=0.6313（32分钟）

**原因**：
- 手工特征工程已提供足够非线性
- 非线性核反而过拟合
- 特征工程 > 算法选择
