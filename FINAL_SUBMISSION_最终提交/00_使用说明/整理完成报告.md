## ⭐ 三大有趣发现

### 1. 频率统计特征的突破

**贡献**: +6.4%性能提升

```
基础5特征:  F1 = 0.5892
+频率3特征: F1 = 0.6271 (+0.0379, +6.4%)
完整37特征: F1 = 0.6587
```

**创新点**: 计算站点/路线被会员使用的比例
- 办公区站点：会员78%
- 旅游景点：散户68%
- 反映城市功能分区对用户类型的影响

### 2. 最小特征集的发现

**关键数据**: 8个特征 = 97.8%性能

```
1特征:   F1 = 0.5672 (86.1%)
8特征:   F1 = 0.6271 (97.8%)
37特征:  F1 = 0.6587 (100%)
```

**实用价值**:
- 适合实时预测（计算快）
- 适合移动端部署（模型小）
- 证明"less is more"

### 3. 线性核优于非线性核

**对比结果**:

| 核函数 | Test F1 | 训练时间 |
|--------|---------|----------|
| LinearSVC | 0.6411 ✅ | 8秒 |
| RBF | 0.6354 | 45分钟 |
| Polynomial | 0.6313 | 32分钟 |

**结论**: 特征工程 > 算法选择
